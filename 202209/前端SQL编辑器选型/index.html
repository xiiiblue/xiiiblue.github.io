前端SQL编辑器选型

Posted on 2022-09-20 Edited on 2022-11-02

基础库

  • code-mirror 著名编辑器基础库，支持SQL，关注
  • monaco-editor 微软，VSCode，有支持SQL的第三方fork

一体化方案

  • phpmyadmin 需要使用PHP二次开发
  • adminer 界面类似phpmyadmin，轻量，使用PHP开发
  • zeppelin 类似JupyterNotebook，数据可视化
  • sqlpad 使用Node开发，关注
  • falcon
  • hue

前端库

  • at.js 输入框自动完成
  • js-sql-parser 前端解析SELECT SQL
  • sql-editor 基于CodeMirror，纯前端，国产个人开发，关注

商业软件

  • sqldev 界面精美，不开源，国产

其它

  • ace 前端JS编辑器，貌似没有对SQL的支持

关于选型

  • 初步了解并罗列，未深入对比
  • 几个一体化的方案，看上去都不太好改
  • 需求灵活的话，有可能要基于code-mirror基础库自已做(或者用sql-editor这种基于code-mirror魔改的版本)
