PHP依赖包学习笔记

Posted on 2021-11-30 Edited on 2022-11-02 Views:

jenssegers/mongodb

  • https://github.com/jenssegers/laravel-mongodb
  • https://www.jianshu.com/p/484fb19c9389
  • https://segmentfault.com/a/1190000018922161

guzzlehttp/guzzle

Guzzle是一个PHP的HTTP客户端，用来轻而易举地发送请求，并集成到我们的WEB服务上。

  • https://docs.guzzlephp.org/
  • https://guzzle-cn.readthedocs.io/zh_CN/latest/

pda/pheanstalk

Beanstalkd 是一个高性能，轻量级的分布式内存队列，C 代码，典型的类 Memcached 设计，协议和使用方式都是同样的风格，所以使用过 memcached 的用户会觉得 Beanstalkd 似曾相识。
pheanstalk是一个操作 beanstalkd 的 php 客户端。

  • https://github.com/pheanstalk/pheanstalk
  • https://www.kancloud.cn/vson/php-message-queue/895573

predis/predis

predis是php连接redis的操作库

  • https://github.com/predis/predis
  • https://segmentfault.com/a/1190000012404974

doctrine/dbal

在Laravel中，使用migration作为数据库的版本控制工具，当需要对已存在的数据表作更改，需要额外引入doctrine/dbal扩展

  • https://github.com/doctrine/dbal
  • https://www.jianshu.com/p/f33db09d3ed6

kittinan/php-crc

Calculates CRC checksums for input data

  • https://packagist.org/packages/kittinan/php-crc

simplesoftwareio/simple-qrcode

Simple QrCode 是基于Bacon/BaconQrCode 开发，适用于Laravel框架的软件包. 我们的目的是让二维码能更加便捷的使用在Laravel框架的项目里

  • https://github.com/SimpleSoftwareIO/simple-qrcode

menthe/aliyun-ueditor

UEditor 是由百度web前端研发部开发所见即所得富文本web编辑器，此包为laravel5的支持,新增多语言配置,可自由部署前端代码

  • https://github.com/menthe/aliyun-ueditor

laravel/passport

Passport提供完整的 OAuth2 服务端实现

  • https://laravel.com/docs/8.x/passport
  • https://learnku.com/docs/laravel/8.x/passport/9420

alibabacloud/client

短信服务（Short Message Service）是阿里云为用户提供的一种通信服务的能力。

  • https://packagist.org/packages/alibabacloud/client
  • https://help.aliyun.com/document_detail/112186.html

jenssegers/agent

laravel-agent 是一个为 Laravel 定制的客户端识别扩展包，基于 Mobile Detect , 提供了非常优雅的接口.

  • https://github.com/jenssegers/agent
  • https://learnku.com/laravel/t/782/extended-recommendation-laravel-user-agent-easily-identify-client-information

dariuszp/cli-progress-bar

Progress bar for cli apps

  • https://github.com/dariuszp/cli-progress-bar

overtrue/laravel-wechat

微信 SDK for Laravel / Lumen， 基于 overtrue/wechat

  • https://github.com/overtrue/laravel-wechat

imangazaliev/didom

HTML 解析器

  • https://github.com/Imangazaliev/DiDOM
  • https://www.muyuanzhan.com/tutorials/php/1030.html
  • https://blog.csdn.net/cunjie3951/article/details/106905467

bvtvd/laravel-gateway-worker

GatewayWorker 是基于 Workerman 开发的一个 TCP 长连接框架，实现了单发、群送、广播等长连接必用的接口

  • https://github.com/bvtvd/laravel-gateway-worker
  • https://learnku.com/articles/25208

fideloper/proxy

如果您的站点位于代理（例如负载均衡器）后面，则您的 Web 应用程序可能存在以下一些问题：
重定向和 PHP 生成的 URL 在其 Web 地址，协议和 / 或端口方面可能不准确。
可能无法为每个用户创建唯一会话，从而导致可能访问不正确的帐户，或者无法让用户完全登录
记录或其他数据收集过程数据可能看起来来自一个位置（代理本身），使您无法区分各个客户端所采取的流量 / 操作。

  • https://github.com/fideloper/TrustedProxy
  • https://learnku.com/docs/the-laravel-way/5.6/Tao-5-6/2947

laravel/tinker

tinker 允许您从交互式 shell 中的命令行与 Laravel 应用程序进行交互

  • https://laravelacademy.org/post/4935.html
  • https://learnku.com/articles/7338/understand-laravel-tinker-shell

laravelcollective/html

Laravel提供了各种内置的标签，以方便，安全地处理HTML表单。所有 HTML 的主要元素都是可以使用Laravel来产生的。

  • https://github.com/LaravelCollective/html
  • https://laravelcollective.com/docs
  • https://segmentfault.com/a/1190000011580448

maatwebsite/excel

Excel 工具包

  • https://github.com/SpartnerNL/Laravel-Excel
  • https://learnku.com/articles/32391

workerman/workerman

workerman是一款开源高性能PHP应用容器

  • https://www.workerman.net/
  • https://github.com/walkor/Workerman
  • https://www.kancloud.cn/walkor/workerman/640361

workerman/mqtt

一个基于workerman的异步mqtt客户端库，可用于接收或者发送mqtt协议的消息

  • https://www.workerman.net/doc/workerman/components/workemran-mqtt.html
  • https://github.com/walkor/mqtt

mrgoon/aliyun-sms

阿里云提供的短信服务

  • https://packagist.org/packages/mrgoon/aliyun-sms
  • https://laravelacademy.org/post/9502

overtrue/laravel-pinyin

Laravel 中文转拼音

  • https://github.com/overtrue/laravel-pinyin
  • https://learnku.com/laravel/t/573/laravel-chinese-pinyin-overtruelaravel-pinyin

zircote/swagger-php

接口API的定义

  • https://github.com/zircote/swagger-php
  • https://segmentfault.com/a/1190000015913371

jenssegers/imagehash

图片摘要

  • https://github.com/jenssegers/imagehash
